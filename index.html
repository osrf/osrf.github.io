  • Robot Software

    ROS (Robot Operating System) provides libraries and tools to help software developers create robot applications. It provides hardware abstraction, device drivers, libraries, visualizers, message-passing, package management, and more. ROS is licensed under an open source, BSD license.

    Visit the ROS website

  • Simulation

    A leader in robotic simulation, Gazebo provides researchers, engineers, and hobbyists a powerful tool in their robot software development toolkit. Through a combination of rigid body physics, sensor models, actuator control, and object modeling Gazebo can recreate a wide variety of robots and environments.

    Visit the Gazebo website
